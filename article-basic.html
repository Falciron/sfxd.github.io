Basic information about the Discord server

Last Updated: June 13, 2018

These are the basic rules and information for the discord server.

Amongst other things, details include all of the bot commands you can use, an explanation of tatsumaki, a suggested format for job offers, etc.

The discord server

To get a standard role, please use one of the following in the #rolerequest channel:

  • +getrole Admin
  • +getrole Dev
  • +getrole Consultant
  • +getrole Marketing Cloud
  • +getrole Commercecloud
  • +getrole Recruiters
Non-standard roles can be requested there by PM'ing @Moderators .
Non-standard roles include but are not limited to Salesforce Internal for example. These require verification.
You can also request to be a Power User or a Moderator at any time. Approval is dependent on the current role holder's decision.

You can have multiple roles.

New to discord ? Learn about formatting your messages, especially for code blocks.

Use any Specialities channel for technical questions. #general can be used for anything including banter and support requests.
#rss_dump is automated.
#random is where memeing takes place.
#nsfw-rant is not really nsfw but will contain strong language.
#job-offers is where you want to speak about job offers. If you're a recruiter, you should head there. While honest mistakes are fine, spamming normal discussion channels with job offers will not be tolerated.

You may use Vexera to listen to music. Simply join a voice channel and use #botchat to request a playlist. While the link in this paragraph gives you all the commands, the most used ones are:

+play $url/query$ Plays a song. Accepts both websites and a search query as a parameter +pause Pauses the song. +clear Clears the queue

Regarding other bots, tatsumaki pretty much handles herself. If someone helped you a lot, you can do !reputation @user to give them a reputation point.

If you REALLY want to see everything Tatsumaki's got to offer, including styling your profile image and such, head over to https://tatsumaki.xyz/commands.html

Note that trying to activate the NSFW commands of the bot is one of the very rare things that will get you instabanned.
